# Security Policy

## Intended Users
This repo is for me, Andy Vo and my professor and it's just for class. I forked the repo from my professor.

## Security Risk Assessment
The repo has code and Jupyter notebooks. If it got into 
the wrong hands, the biggest concern would probably be other students copying 
the work, which is more of an academic issue than a real security threat. 
The data in here was given to us by the professor so it's not like it's sensitive 
or personal information. Overall the risk is pretty low since this is just a 
class assignment and nothing here is being deployed anywhere.

## Security Measures
This repo was forked from my professor's repo so I didn't really set anything 
up myself. There's a .gitignore file in place to avoid committing any sensitive/unnecessary files, a CODEOWNERS file has been added to designate ownership of the repo, and there's no API keys or credentials are hardcoded anywhere in the code that I know of.
